# homelab-backup

Simple scripts for backing up, restoring, and verifying homelab backups.

## Project Structure

- `backups/`: Backup destination placeholder.
- `logs/`: Runtime logs.
- `backup.sh`: Main backup script.
- `restore.sh`: Restore backups.
- `verify.sh`: Verify backups.
- `config.sh`: User configuration.

## Getting Started

```bash
chmod +x backup.sh restore.sh verify.sh
./backup.sh
```
